Visualisation COVID-19 par département
  • Hospitalisés/Population
  • Hospitalisés/Lits
  • Réanimations/Population
  • Réanimations/Lits
Les visualisations présentent quatres indicateurs nécéssaires afin de suivre l'évolution du Covid-19 par département et par jour en France. Pour les réaliser, j'ai préparé un workflow de données sur l'outil Dataiku, qui récupèrent les données à jour à partir du siteweb data.gouv.fr puis fait des traitement afin de récupérer les indicteurs voulus.
Ces derniers sont les suivants :
  • Hospitalisés/Population
    • log ( 1 + Nombre de personnes hospitalisés cumulé / le nombre de personne du département )
  • Hospitalisés/Lits
    • log ( 1 + Nombre de personnes hospitalisés cumulé / le nombre de lits dans les hopitaux du département* )
  • Réanimations/Population
    • log ( 1 + Nombre de personnes en réanimation cumulé / le nombre de personne du département )
  • Réanimations/Lits
    • log ( 1 + Nombre de personnes en réanimation cumulé / le nombre de lits dans les hopitaux du département* )
